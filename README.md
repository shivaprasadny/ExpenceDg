# ExpenseDg

A local-first expense tracker built with Expo + React Native.

## Features
- Offline-first (no internet required)
- Local SQLite database
- Optional App Lock (4-digit PIN)
- Mandatory security questions for recovery
- Change PIN & disable lock
- Expo Router architecture

## Tech Stack
- Expo (React Native)
- Expo Router
- SQLite
- SecureStore
